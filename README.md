# 💫 About Me:
Soy Ingeniero de Sistemas con formación en el desarrollo de software, gestión de bases de datos y análisis de datos. Me apasiona la resolución de problemas a través de la tecnología, aplicando buenas prácticas de programación, control de versiones con Git y metodologías modernas de desarrollo. Tengo experiencia trabajando con Python, SQL, Power BI y herramientas de visualización, además de un interés constante en aprender nuevas tecnologías y fortalecer mis habilidades en áreas como el desarrollo web, machine learning y la ciencia de datos.
